# 📝 Notes App

A simple Notes App built using HTML, CSS and JavaScript.

## 🚀 Features

- Create notes
- Edit notes
- Delete notes
- Notes are saved using Local Storage
- Responsive design
- Delete confirmation
- Input validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
